Без серверної адмін-функції можна безпечно міняти тільки свій гаманець.
1) Мій гаманець (wallets)
Працює з RLS: UPDATE дозволений лише для user_id = auth.uid().
2) Користувачі (потрібен бекенд)
З браузера з anon-key не можна безпечно читати список користувачів або міняти чужі баланси (це потребує service-role / Edge Function). UI нижче — заготовка під Edge Function (коли підключиш).
Поки що натискання просто пише в LOG, бо немає endpoint.
